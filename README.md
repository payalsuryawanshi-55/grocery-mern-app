# 🛒 Grocery MERN App

A full-stack Grocery Shopping Web Application built with the MERN Stack.

## Features
- JWT Authentication
- Product Management
- Shopping Cart
- Order Management
- Seller Dashboard
- Cloudinary Image Upload
- Responsive UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- Cloudinary

## Installation

```bash
cd backend
npm install

cd ../client
npm install
```
